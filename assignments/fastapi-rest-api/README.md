# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective
Learn how to build and deploy REST APIs using the FastAPI framework in Python. Students will create endpoints, handle requests, and return responses, practicing modern web development skills.

## 📝 Tasks
- Task 1: Set up a FastAPI project and create a basic endpoint that returns a welcome message.
- Task 2: Add endpoints to handle CRUD operations for a simple resource (e.g., items or users).
- Task 3: Implement input validation and error handling using FastAPI features.
- Task 4: Test your API using HTTP requests (with tools like curl or Postman).

## Requirements
- Use FastAPI to build the API
- Include endpoints for Create, Read, Update, and Delete operations
- Validate input data and handle errors gracefully
- Provide clear instructions for running and testing the API

## Resources
- starter-code.py (provided)
- FastAPI documentation: https://fastapi.tiangolo.com/
- Example tools: curl, Postman

## Student Guidance
- Review the starter code for structure and hints
- Ask questions if you get stuck or need clarification
- Experiment with different endpoints and data
