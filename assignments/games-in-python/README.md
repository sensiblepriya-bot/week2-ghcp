
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic Hangman word-guessing game in Python. Practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Task 1: Game Setup

#### Description
Create the basic structure for the Hangman game. Define a list of possible words and set up the game loop.

#### Requirements
Completed program should:
- Randomly select a word from a predefined list
- Initialize variables for guesses and attempts
- Start the main game loop

### 🛠️ Task 2: Gameplay Logic

#### Description
Implement the logic for accepting guesses, updating progress, and tracking incorrect attempts.

#### Requirements
Completed program should:
- Accept letter guesses from the player
- Show current progress using underscores and revealed letters (e.g., _ a _ _ m a n)
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Display win/lose messages
